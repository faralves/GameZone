-------------------------------Gerar certificado para rodar em localhost--------------------------

instalar o openssl
https://slproweb.com/products/Win32OpenSSL.html

ou se tiver o gitHub instalado ele já possui o openssl

rodar os seguintes comandos:
1º acessar o diretório onde está o certificado, exemplo:
cd C:/Projetos/Fiap/Grupo31/TechChalenge/Fase2/GameZone/docker/certs

Crie um Certificado Autoassinado:
# Gere uma chave privada
openssl genpkey -algorithm RSA -out private-key.pem

# Gere uma solicitação de certificado (CSR)
openssl req -new -key private-key.pem -out cert-request.csr

# Gere o certificado autoassinado usando a chave privada e o CSR
openssl x509 -req -days 365 -in cert-request.csr -signkey private-key.pem -out certificate.pem

Converta o Certificado para Formato PFX:
openssl pkcs12 -export -out certificate.pfx -inkey private-key.pem -in certificate.pem

inserir a senha do certificado.
